## 💻 عن المشروع
الموقع عبارة عن معرض أعمال شخصي متكامل ومتجاوب، يعرض:
* **المشاريع (Projects)**: تصميم وتطوير تطبيقات ومواقع (مثل: نقطة، مقود، فينلانس، قصد، نسق).
* **الشهادات (Certifications)**: شهادات المعسكرات التدريبية (مثل معسكر تصميم تجربة المستخدم من أكاديمية طويق، ومعسكر وجهة التقني من جامعة الملك سعود).
* **المهارات (Skills)**: مهارات التصميم، البحث، هندسة المعلومات، والبرمجة.
* **الخبرات والتعليم (Experience & Education)**.

---

## 🛠️ التقنيات المستخدمة
* **HTML5**: لهيكلة صفحات الموقع بشكل دلالي (Semantic HTML).
* **CSS3**: للتنسيق، وتصميم الهوية البصرية الأنيقة، والتجاوب مع جميع الشاشات (Responsive Design).
* **JavaScript**: لإضافة التفاعلية مثل تصفية وتصنيف المشاريع (Filter Projects) وتأثيرات الحركة.

---

## 📂 هيكلة ملفات المشروع
```text
├── index.html          # الصفحة الرئيسية لمعرض الأعمال
├── projects.html       # صفحة تفاصيل وعرض كل المشاريع مع نظام التصفية
├── styles.css          # ملف التنسيقات والألوان والهوية البصرية
├── script.js          # ملف برمجة التفاعلات والحركات
├── assets/             # مجلد يحتوي على الصور، اللوقو، والشهادات المحدثة
└── PRD.md              # وثيقة متطلبات المنتج (Product Requirements Document)
```

## 🚀 كيفية تشغيل المشروع محلياً
1. قم بتحميل المستودع (Clone) أو تنزيله كملف مضغوط (ZIP).
2. افتح ملف `index.html` في أي متصفح ويب (مثل Chrome أو Safari).
